Tietoa minusta


Olen nykyään Espoossa asuva juuriltani joensuulainen vuonna 1990 syntynyt kahden lapsen isä. Aikaisemmin opiskelin tietojenkäsittelytiedettä Itä-Suomen yliopistossa, mutta Espooseen muuttamisen seurauksena päädyin Haaga-Helian tietojenkäsittelytieteen pariin. Opintojeni ohella työskentelyn täysaikaisesti Terveystalolla. Kiinnostukseni kohteita ovat muun muassa ohjelmointi, videopelit sekä politiikka. Pitkässä juoksussa tavoitteeni on päästä tekemään työtä, mielelään ohjelmistokehityksen saralla, jolla on positiivista yhteiskunnallista merkitystä.

Ohjelmoinnista ja tietotekniikasta yleisesti olen ollut kiinnostunut jo pikkulapsesta asti. Haaga-Helian tietojenkäsittelytieteiden tradenomin tutkinto tulee olemaan ensimmäinen suoritettu tutkintoni, vaikka alaa olenkin opiskellut aikaseimmin. Erikoistun backend-ohjelmointiin ja tutuin ohjelmointikieli on Java. Pyrkimykseni on kehittyä Full Stack -kehittäjäksi. Muita osaamiani kieliä (vähintään perusteet) ovat esimerkiksi JavaScript, HTML, CSS, Python ja SQL. Erilaisia ohjelmoinnin työkaluja sekä rajapintoja olen käyttänyt runsaasti, esimerkkeinä Spring Framework, NodeJS, MongoDB, Azure DevOps sekä erilaiset IDE:t kuten Visual Studio Code ja Eclipse.
